# GODS ZEAL XMD - Session Generator

## Quick Start

1. **Create a Mega.nz account** at https://mega.nz
2. **Update `mega.js`** with your Mega email and password
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Deploy to Render, Railway, or any Node.js host

## Features
- ✅ Pair Code generation
- ✅ QR Code generation  
- ✅ Auto newsletter follow
- ✅ Auto group join
- ✅ Session file delivery via WhatsApp
- ✅ Auto cleanup after pairing

## Deploy to Render

1. Push this folder to a GitHub repo
2. Go to https://render.com → New Web Service
3. Connect your repo
4. Set Build Command: `npm install`
5. Set Start Command: `npm start`
6. Deploy!
